<h1>Panopto Integration with Angel</h1>
<b><u>What is it?</u></b>

Panopto's integration with Angel to allow shared login and course management between systems. The code supports Angel version 7.3, 7.4, and 8.0. The folder named PanoptoConnector is for version 7.4.

<b><u>Credits</u></b>

This was built by Panopto

<b><u>How can I help?</u></b>

Fork the block, fix a bug or add a new feature, and send us a pull-request. Or, if you're not a developer, but you've found a bug, add it to our issue tracker.
